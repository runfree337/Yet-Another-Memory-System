**Decision**: YAMS is the product. The framework lives at the repository root (no `_meta/`
wrapper), stands alone from its original host project (embedded `adapters/`, no outbound
references to host artifacts), and is written in English — prose and machine vocabulary alike.

**Why**: the framework was extracted from a host game project and still carried its packaging
("out-of-scope meta folder"), its references (`.claude/` hooks, host decision ids) and its
language. A standalone framework cannot ask adopters to read another private repository.
Alternative ruled out: keeping `_meta/ai-workflow/` with fixed READMEs — simpler, but the
"meta" packaging would remain a vestige contradicting the repository's purpose.

**Invariant**: no document or script in this repository references an artifact that does not
ship with it (verifiable: `checks/doc-refs-check.py` at zero).
